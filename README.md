# UAE Banking Hub 🏦

A comprehensive UAE banking intelligence platform covering **all 60+ CBUAE-licensed banks** — national, foreign, and digital.

## Features

- **Compare Products**: Side-by-side comparison of savings accounts, credit cards, personal loans & auto finance
- **Bank Profitability**: FY 2025 net profit rankings with YoY growth, total assets, ROE
- **Live News**: Real-time UAE banking & finance news (powered by Anthropic API)
- **Full Bank Directory**: All CBUAE-licensed banks — searchable by name, type, category
- **Mobile-First**: Designed for phones, works everywhere

## Banks Covered

- **22 National Banks**: Emirates NBD, FAB, ADCB, DIB, Mashreq, ADIB, RAKBANK, CBD, Al Hilal, Ajman Bank, Emirates Islamic, Bank of Sharjah, SIB, UAB, NBF, CBI, InvestBank, NBQ, NBAD (merged into FAB), Wio, Zand, Emirates Investment Bank
- **28+ Foreign Banks**: HSBC, Standard Chartered, Citibank, BNP Paribas, Deutsche Bank, Barclays, Bank of China, Credit Agricole, and more
- **Digital Banks**: Wio Bank, Zand Bank, Mashreq Neo, Liv by ENBD

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Deployed on Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push to GitHub
2. Import project at [vercel.com/new](https://vercel.com/new)
3. Deploy — zero config needed
